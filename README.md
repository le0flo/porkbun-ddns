# Porkbun DDNS

Self hosted dynamic domain script that connects with your [porkbun](https://porkbun.com) account and updates the subdomain with the current ip.

### Usage

Download the repo:

```sh
git clone https://github.com/le0flo/porkbun-ddns
cd porkbun-ddns
```

Run the container:

```sh
docker compose up
```
